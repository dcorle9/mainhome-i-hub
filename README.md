# mainhome-i-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，主要用于存放各类静态页面资源，方便统一管理与访问。

## 项目简介

本仓库以目录方式组织多个独立的 HTML 页面，每个页面均可单独使用。所有页面均为静态内容，不依赖后端服务，适合用于归档、展示或临时发布。

## 目录说明

```
mainhome-i-hub/
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── styles/
    ├── scripts/
    └── images/
```

- `index.html`：仓库入口页面，可链接到各归档页面。
- `pages/`：每个子页面独立存放，命名清晰，便于直接访问。
- `assets/`：存放页面间共享的样式、脚本与图片资源。

## 页面归档说明

- 每个 HTML 页面均为完整自包含文件，或仅依赖 `assets/` 目录下的公共资源。
- 归档页面可能来源于不同项目或用途，彼此之间无直接关联。
- 页面内容保持原始状态，仅做必要的路径调整以确保在仓库中正常运行。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务发布后在线访问。
